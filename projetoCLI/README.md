# 🎧 Music Picker CLI  
A tiny command‑line helper for people who love music but hate choosing what to play.

## 🌱 Why this exists
I’m always overwhelmed by the sheer amount of music and albums available.  
Every time I want to listen to something new, I freeze, overthink, and end up looping the same handful of albums forever.

So I built this little CLI app to break that cycle.

## 🎲 What it does
- Randomly selects an album or track from a list you provide  
- Helps you avoid decision paralysis  
- Encourages rediscovering forgotten favorites  
- Makes listening fun again

## 🛠 How it works
1. Add your albums, artists, or playlists to a simple data file  
2. Run the CLI  
3. Get a random pick instantly  
4. Press play and enjoy without overthinking

## 🚀 Why it helps
This tool removes the burden of choice.  
Instead of scrolling endlessly through your library, you get a single, clean suggestion — enough to get you listening again.

## 📌 Dependencies
- Typer
- Uv

## 💡How to use
- Open Container with Uv (astral)
- uv add typer
- source .venv/bin/activate
- python src/cli.py

  Then it will open a small menu to choose what to do
